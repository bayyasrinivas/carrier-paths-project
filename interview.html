  • Find the two non-repeating elements in an array of repeating elements - Accolite, Amazon, FactSet, Google, MakeMyTrip, Microsoft, Qualcomm, Samsung
  • Program to find whether a number is a power of two - Adobe
  • Find position of the only set bit - Microsoft
  • Count number of bits to be flipped to convert A to B - MAQ Software
  • Count total set bits in all numbers from 1 to n - Microsoft
  • Copy set bits in a range - Facebook
  • Calculate square of a number without using *, / and pow() - Amazon
  • Divide two integers without using multiplication, division and mod operator - Microsoft
  • Power Set - Google, Adobe, Paytm
